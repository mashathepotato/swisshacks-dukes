# Context

Shared context for the SwissHacks "Dukes" team. Keep this folder up to date so
anyone (human or AI agent) can get oriented fast.

## Files

- `challenge.md` — the challenge prompt, rules, judging criteria, deadlines.
- `project.md` — what we're building: idea, scope, MVP, stretch goals.
- `architecture.md` — stack, components, data flow, key decisions.
- `team.md` — who's who and who owns what.
- `decisions.md` — running log of decisions and why we made them.

## Conventions

- One topic per file; link between files rather than duplicating.
- Convert relative dates ("tomorrow") to absolute ones.
- When something changes, update the file rather than appending stale notes.
